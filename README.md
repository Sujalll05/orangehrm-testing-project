# OrangeHRM Manual Testing Project

## Project Overview

I recently completed a Manual Testing Project on the OrangeHRM web application. The objective of this project was to apply software testing concepts in a real-world application and gain hands-on experience in manual testing.

During this project, I performed Functional Testing, Exploratory Testing, and Regression Testing across multiple modules of the application.
